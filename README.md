# 🚀 Subhadip Pal — Personal Portfolio Website

> A recruiter-impressive, fully responsive personal portfolio website built with pure HTML, CSS & JavaScript.

**Live Demo:** [palsubhadip304-ship-it.github.io/portfolio](https://palsubhadip304-ship-it.github.io/Personal-Portfolio-website-/)

---

## ✨ Features

- **Dark elegant design** — Navy blue + orange accent color palette
- **Custom animated cursor** — Interactive cursor with ring effect
- **Smooth scroll animations** — Elements reveal as you scroll
- **Animated skill bars** — Trigger on viewport entry
- **Fully responsive** — Works on mobile, tablet & desktop
- **Single HTML file** — No frameworks, no dependencies

## 📄 Sections

| Section | Content |
|---|---|
| Hero | Name, tagline, CTA buttons, social links |
| About | Bio, education info, achievement stats |
| Skills | Data analytics, Excel, AI tools with animated bars |
| Experience | Zeyras Agrotech internship timeline |
| Projects | 4 data analytics projects with tags |
| Certifications | Statistics Mastery, Time Management, Internship |
| Contact | Contact links + message form |

## 🛠️ Tech Stack

```
HTML5  ·  CSS3  ·  Vanilla JavaScript
Google Fonts: Playfair Display + DM Sans + DM Mono
IntersectionObserver API for scroll animations
```

## 📁 File Structure

## 🚀 Deploy on GitHub Pages

1. Go to repo **Settings → Pages**
2. Source: **Deploy from branch → main → / (root)**
3. Click **Save**
4. Live at: `https://palsubhadip304-ship-it.github.io/portfolio`
